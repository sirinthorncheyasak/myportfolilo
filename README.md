# Portfolio Site

## Overview
This project is a personal portfolio site that showcases various projects and skills. It is designed to be responsive and user-friendly, providing an engaging experience for visitors.

## File Structure
```
portfolio-site
├── index.html        # The main HTML file for the portfolio site
├── styles            # Directory containing CSS styles
│   └── styles.css    # The CSS file for styling the portfolio site
├── scripts           # Directory containing JavaScript files
│   └── script.js     # The JavaScript file for interactivity
├── README.md         # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine.
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory.
   ```
   cd portfolio-site
   ```
3. Open `index.html` in your web browser to view the portfolio site.

## Features
- **Responsive Design**: The site is designed to work on various screen sizes, ensuring a good user experience on both desktop and mobile devices.
- **Project Filtering**: Users can filter projects by category (e.g., web, mobile, data) using buttons that dynamically update the displayed projects.
- **Hamburger Menu**: A collapsible navigation menu for mobile devices that enhances usability.
